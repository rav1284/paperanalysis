# Intelligent Research Paper Analysis Chatbot

## Features
- PDF Upload & Text Extraction
- Text Summarization (TextRank)
- Question Answering (TF-IDF + Word2Vec)
- Plagiarism Detection
- PPT Generation
- Streamlit UI

## Tech Stack
- Python
- NLTK
- Scikit-learn
- NetworkX
- Gensim (Word2Vec)
- NumPy
- PyPDF2
- python-pptx
- Streamlit

## Run Locally
```bash
pip install -r requirements.txt
streamlit run app.py
